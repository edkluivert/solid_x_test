## 1.0.1

* Initial release of `solid_x_test`.
* Includes `solidTest` utility for testing `Solid` ViewModel streams and interactions.
* Includes `MockSolid` base class for generating ViewModel mocks in Widget tests using `mocktail`.
* Includes `mutationTest` utility and `MutationState` matchers (`isMutationLoading`, etc.) for testing `solid_x` Mutations.
* Includes `MockMutation` base class for generating Mutation mocks.
